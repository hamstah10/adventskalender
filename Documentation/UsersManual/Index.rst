.. _users-manual-create-doors:

Creating Doors
==============

1. Go to the :guilabel:`List` module
2. Select the folder where you want to store the doors
3. Click :guilabel:`Create new record`
4. Select :guilabel:`Adventskalender TÃ¼rchen`
5. Fill in the following fields:

   * **Day (1-24)**: The door number (required)
   * **Title**: Door headline (required)
   * **Description**: Short description
   * **Content**: Main content (Rich Text Editor)
   * **Image**: Upload an image
   * **Video**: Upload a video file
   * **Audio**: Upload an audio file
   * **Link**: External URL

6. Click :guilabel:`Save`

.. _users-manual-create-voucher:

Creating Vouchers
=================

1. Edit a door record
2. Go to the :guilabel:`Gutschein` tab
3. Click :guilabel:`Create new`
4. Fill in:

   * **Headline**: Voucher title (e.g., "Geschenkgutschein")
   * **For (Name)**: Recipient name
   * **Description**: What the voucher is for
   * **From (Name)**: Sender name
   * **Design**: Choose between "Klassisch" or "Weihnachtsmann"

5. Click :guilabel:`Save`

.. _users-manual-custom-door-design:

Custom Door Design
==================

Each door can have an individual design:

1. Edit a door record
2. Go to the :guilabel:`Design` tab
3. Enable :guilabel:`Individuelles Design aktivieren`
4. Configure:

   * **Color Start**: Gradient start color
   * **Color End**: Gradient end color
   * **Border Color**: Border color
   * **Border Width**: Border width in pixels (0-10)

5. Click :guilabel:`Save`

.. _users-manual-insert-plugin:

Insert Plugin
=============

1. Go to the :guilabel:`Page` module
2. Select the page where you want the calendar
3. Click :guilabel:`+ Content`
4. Select :guilabel:`Plugin`
5. Choose :guilabel:`Adventskalender`
6. In the :guilabel:`Plugin` tab:

   * Select **Record Storage Page** (folder with your doors)
   * Optional: Set **Custom Title**
   * Optional: Set **Custom Description**
   * Select **Background Image**

7. Click :guilabel:`Save`

.. _users-manual-translations:

Translations
============

To translate door records:

1. Edit a door record
2. Go to the :guilabel:`Language` tab
3. Select target language
4. Click :guilabel:`Translate`
5. Edit the translated fields
6. Save

The same applies to voucher records.
