===========
For Editors
===========

.. _editor-quick-start:

Quick Start
===========

1. Create 24 door records (one for each day)
2. Add content, images, or vouchers to each door
3. Insert the plugin on a page
4. Select the folder containing your doors

.. _editor-best-practices:

Best Practices
==============

Door Titles
-----------

Keep door titles short and engaging:

* ❌ "This is the door for December 1st with a special surprise"
* ✅ "Nikolaus-Überraschung"

Door Content
------------

Use the Rich Text Editor to format your content nicely:

* Add images
* Use headings
* Add links
* Format text (bold, italic)

Vouchers
--------

When creating vouchers:

* Use clear, descriptive text
* Keep the description concise
* Choose the appropriate design (Classic or Santa)
* Translate vouchers for multi-language sites

.. _editor-tips:

Tips & Tricks
=============

Special Days
------------

* Day 6 (Nikolaus) automatically gets special styling
* Day 24 (Christmas Eve) automatically gets gold styling
* Override these with custom door designs if needed

Background Images
-----------------

Choose background images that:

* Have good contrast with the glass doors
* Are high resolution (min 1920x1080)
* Match the Christmas theme

Music
-----

* Keep background music at moderate volume
* Use copyright-free music
* Provide a toggle so users can turn it off
